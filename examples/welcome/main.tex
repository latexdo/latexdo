\begin{abstract}
TeXly brings a modern code editor, fast local compilation, useful diagnostics,
and a PDF preview into one calm workspace. Edit this file and press
\textbf{Compile} to see your changes.
\end{abstract}

\section{Write without friction}

The editor understands LaTeX commands, keeps your project files close, and
builds with the same trusted TeX toolchain used by professional workflows.

\begin{itemize}
  \item Monaco-powered editing
  \item One-click \texttt{latexmk} compilation
  \item Automatic bibliography and reference passes
  \item Build errors linked back to source lines
\end{itemize}
